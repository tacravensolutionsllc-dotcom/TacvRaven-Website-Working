Everything you need to know about Black Feather and Corvus Engine

Air-gapped means Black Feather operates entirely offline with zero network connectivity required. All analysis happens locally on your machineno data is ever transmitted to external servers, cloud services, or the internet. This makes it ideal for classified environments, SCIFs, and organizations with strict data sovereignty requirements.

Black Feather is delivered as a single-file HTML application that runs in any modern web browserno installation, dependencies, or administrator privileges required. Simply open the file and start analyzing. Corvus Engine follows the same portable deployment model for log processing and data conversion.

All licenses include documentation and email support. Enterprise licenses include priority support with guaranteed response times and dedicated account management. We also offer custom development services for organizations with unique requirements.
